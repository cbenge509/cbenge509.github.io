## Maintenance

### Adding New Projects

1. Create folder: `mkdir projects/newproject`
2. Create page: Copy existing project `index.html`
3. Add thumbnail: `assets/img/newproject_project.png`
4. Add card to `projects/index.html`
5. Commit and push

### Updating Content

- Edit HTML files directly
- Replace images in `/assets/img/`
- Replace PDFs in `/assets/pdf/`
- Commit and push to deploy
